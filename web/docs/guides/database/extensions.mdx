拡張とは、その名の通り、Postgresのコア部分ではない機能でデータベースを「拡張」するものです。
Supabaseには、最も便利なオープンソースの拡張がいくつかプリインストールされています。

SQLコードが`create extension`（拡張を作成）していても、これは「拡張を有効にする」と同じことです。
拡張を無効にするには、`drop extension`を呼びだします。

Enabling few extensions by the SQL code `create extension <extension-name> with schema extensions` may lead to permission issues (e.g. `dblink`, `http`, `pg_cron`).
